# STEM Hub

STEM Hub is an open-source STEM revision platform for CAIE and ZIMSEC curricula. It provides comprehensive resources for students to excel in their STEM subjects.

## Features

- **Subject Directory**: Browse subjects by discipline, board, and level
- **Comprehensive Notes**: Access peer-written notes for all STEM subjects
- **Interactive Quizzes**: Test your knowledge with quick quizzes
- **User Profiles**: Create an account to bookmark your favorite subjects and topics
- **Dark Theme**: Enjoy a dark-themed interface for comfortable studying
- **Responsive Design**: Access the platform on any device

## Tech Stack

- **Frontend**: Next.js, React, Tailwind CSS
- **Authentication**: Supabase Auth
- **Database**: Su
